# tommysstabsuebung

LageSprint ist ein kurzweiliges Karteikarten-Lernspiel fuer die Stabsausbildung.

## Starten

```powershell
cd karteikarten-spiel
npm install
npm run dev
```

Die App laeuft danach lokal unter `http://127.0.0.1:5173`.

## Build

```powershell
cd karteikarten-spiel
npm run build
```

Die Karteikarten werden beim Build aus dem XML-Export importiert.
